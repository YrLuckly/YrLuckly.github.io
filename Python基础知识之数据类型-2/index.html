Python基础知识之数据类型(2)

本次文章会讲到数据类型中的字符串、字典、列表、元祖、集合。在以后的工作中都会用到。

1.字符串—str

Python中凡是用引号引起来的数据可以称为字符串类型，组成字符串的每个元素称之为字符，将这些字符一个一个连接起来，然后在用引号起来就是字符串。所有标准序列操作都适用于字符串，但是字符串是不可变的，所以，所有的元素赋值和切片赋值都是不可取的

w = '双击666'   # 对w这个字符串来说，它由‘双，击，6，6，6’五个字符组成
1.1 字符串的索引与切片

组成字符串的字符从左往右，依次排列，从左往右依次对应：0,1,2,3…,其中必须注意首位第一是0。

1.1.1 索引

索引也是下标，就是字符串组成的元素从第一开始，初识索引为0 。
举例说明：

s = 'abcdefg'

print(s[0])       # a
print(s[1])       # b
print(s[5])       # f
print(s[6])       # g
1.1.2 切片

切片就是通过索引（索引：索引：步长）截取字符串的一段，形成新的字符串（原则就是顾头不顾尾）。

s = 'abcdefg'

print(s[0:2])       # abc  其中0可以默认不写
print(s[1:5])       # bcdef
print(s[:])         # 直接到最后
print(s[:-1])       # abcdef  -1值最后一个元素的索引。
print(s[1:5:2])       # bd

1.2 format 格式化输出

除了上篇文章介绍的%d的格式化输出方法外，现在有关于format的格式化输出也可以看作替换

1.2.1 format三种方式

1.

s1= "name{0},age{1},hobby{2}".format('老左','24','女票女昌')

print(s1) # name老左,age24,hobby女票女昌

2.

s1 = "name{},age{},hobby{}".format('老左','24','女票女昌')

print(s1)       # name老左,age24,hobby女票女昌

3.

s1 = "name{name},age{age},hobby{hobby}".format(name='老左',age= '24',hobby='女票女昌')

print(s1)       # name老左,age24,hobby女票女昌
1.3 字符串常用方法

字符串除了切片，还有一些其他的方法，这些方法中有些会常用到

1.3.1 capitalize ***

方法capitalize将首字母大写其余字母小写

s = 'abcdefg'
s1 = s.capitalize()
print(s1)            #  Abcdefg
1.3.2 upper、lower ***

方法upper是全部大写，方法lower是全部小写

s = 'abcdefg'
s1 = s.upper()
print(s1)            #  ABCDEFG

s = 'ABCDEFG'
s1 = s.upper()
print(s1)            #  abcdefg
1.3.3 startswith、endswith ***

方法startswith是判断以什么为开头,方法endswith是判断以什么为结尾

s = 'abcdefg'
s1 = s.startswith('a')
print(s1)            #  True

s = 'abcdefg'
s1 = s.endswith('g')
print(s1)            #  True
1.3.4 find、index ***

方法find：通过元素找索引，可切片，找不到返回-1
方法find：通过元素找索引，可切片，找不到报错

s = 'abcdefg'
s1 = s.find('a')
print(s1)  # 0

s2 = s.find('t')
print(s2)  # -1

s1 = s.index('a')
print(s1)  # 0

s2 = s.index('r')
print(s2)  # ValueError: substring not found
1.3.5 strip ***

方法strip是去除前后两边的空格、换行符、制表符

s = '  abcdefg  '
s1 = s.strip()
print(s1)  # abcdefg
1.3.6 split ***

方法split是将字符串转变成列表

s = 'abcdefg'
s1 = s.split()
print(s1)      # ['abcdefg']
1.3.7 join ***

方法join是 将列表变成字符串

s = ['a','b','c']
s1 = '+'.join(s)
print(s1)         # a+b+c
1.3.8 center *

方法center是居中

s = 'abcdefg'
s1 = s.center(10)
print(s1)            #   abcdefg  两边空格补充
1.3.9 swapcase *

方法swapcase是大小写翻转

s = 'abcdefgH'
s1 = s.swapcase()
print(s1)            #  ABCDEFGh
1.3.10 title

方法title是非字母隔开的每个单词的首字母大写

s = 'a1bc3de5fg'
s1 = s.title()
print(s1)        # A1Bc3De5Fg
1.3.11 replace
s = 'abc'
s1 = s.replace('a','222')
print(s1)                 # 222bc
1.4 判断字符串是否满足特定条件

很多字符串方法都以is开头，如isspace、isdigit和isupper，它们判断字符串是否具有特定的性质。如果这些字符串具备特定的性质，返回True，否则返回False。

isalnum,isalpha,isdecimal,isdigit,isidentifier,islower,is,numeric,isprintable,isspace,istitle,isupper
